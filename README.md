# Wayfinder

> 帮你逐渐看见「你是谁，以及你可以成为谁」的系统。

**Wayfinder** 是一个个人探索与成长系统。它帮助你理解自己、探索世界中可能感兴趣的方向，并在你选定某个方向后，持续深入地学习与成长。

由三个协作的 Claude 技能（skills）组成，数据在本地按契约流转。

---

## 🌱 这是什么？

大多数人的问题，不是「缺信息」，而是：

- 不知道自己真正感兴趣什么
- 不知道自己适合什么方向
- 不知道下一步该往哪里走

Wayfinder 试图解决的就是这个问题。

它不是聊天工具，不是测评工具，也不是内容推荐器。它是一个：

> 持续运行的人生探索与成长系统

---

## 🧠 核心思想

Wayfinder 由三个系统组成，各回答一个问题：

### 1️⃣ 画像系统 · selfport

> 你是谁？

通过播客式长期访谈，逐步构建你的结构化画像：

- 成长经历（人生时间线）
- 兴趣信号
- 价值观
- 优势与能力
- 当前人生阶段
- 关系画像
- 思维方式与学习风格

输出不是标签，而是一个持续更新的**自我模型**。

### 2️⃣ 探索系统 · exploration

> 世界上有什么你可能会感兴趣的东西？

读你的画像，抽象出你的「兴趣 DNA」，然后推出**可探索的方向 + 一个当下最值得先看的推荐 + 轻量探索任务**。

它的定位是**人生地图师**——不替你决定去哪，只帮你看见：自己在哪、世界有多大、下一步可以往哪走。

> 把「未知世界」变成「可探索结构」

### 3️⃣ 学习系统 · teach

> 如果你选定一个方向，如何真正学进去？

接过你选的方向，生成一份学习 mission，然后基于「最近发展区」排互动课程，一节一节带你往前走。

> 把「兴趣」转化为「能力」

---

## 🔁 系统循环

Wayfinder 是一个持续循环的系统：

```
画像（你是谁）
   ↓  只读
探索（你可能对什么感兴趣）
   ↓  handoff：方向 + 痛点
学习（你如何深入）
   ↓  反馈回写
更新画像（你变成了谁）
```

每一轮循环，你都会更清楚：自己、世界、未来的可能性。

---

## 🧭 它和传统工具的区别

**❌ 传统方式**
- 人格测试 → 给你一个标签
- 推荐系统 → 推给你内容
- 学习平台 → 让你学课程

这些是「静态系统」。

**✅ Wayfinder 的方式**
- 画像是动态生长的
- 兴趣是演化的
- 学习是按你定制的路径
- 人生是可建模的

它不直接给你答案，而是：

> 帮你逐渐看清「问题本身是什么」

---

## 🧩 系统输出

在过程中，你会逐步得到：

- 一份**持续更新的人生画像**
- 一张**可探索的方向地图**（不是推荐清单，是「你可能对什么」的空间结构）
- **一个**当下最值得先看的方向 + 为什么是你
- 一系列**轻量探索任务**（驱动你真正迈出去）
- 选定方向后：一份**学习 mission** + 互动课程

## 📦 安装

把 `skills/` 下的三个目录复制进 Claude Code 技能目录：

```bash
# Unix / macOS
cp -r skills/* ~/.claude/skills/

# Windows (Git Bash)
cp -r skills/* "/c/Users/<你>/.claude/skills/"
```

然后：`/selfport` 建画像 → `/exploration` 探索 → `/teach` 学习。
