# Session 2：兴趣与好奇心

> 本文件是访谈者的「地图」，不是照念的剧本。访谈哲学、硬规则、挖掘技巧见 `../SKILL.md`。

## 目标
发现兴趣信号。

## 开场白（破冰）
> "平时不上班 / 不忙的时候，你一般都干嘛？刷手机也算。"

从具体的、低门槛的行为切入（行为反推），别上来就问抽象的"你的兴趣是什么"。

## 核心问题（地图，按需自然带入，绝不连续抛出）
- 最近一年最关注什么？
- 最喜欢哪些书？
- 最喜欢哪些播客？
- 最喜欢哪些 UP 主？
- 最喜欢哪些人物？
- 什么内容让你停不下来？

单轮只问一个；用户答不上来时，切到**行为反推**（"最近一次让你熬夜看的是什么"）。

## 覆盖检查清单
- [ ] 关注的领域 / 话题
- [ ] 书
- [ ] 音频 / 视频（播客、UP 主）
- [ ] 人物
- [ ] "停不下来"的内容（最强信号）

## 收尾合成
1. **概览**（叙事）："你的兴趣里有一条挺明显的线是……"
2. **结构化信号**：兴趣关键词
3. **确认**后，套 `../templates/INTEREST_SIGNALS.md` 写入。

## 输出
`INTEREST_SIGNALS.md`
