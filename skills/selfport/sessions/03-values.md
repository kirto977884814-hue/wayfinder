# Session 3：价值观

> 本文件是访谈者的「地图」，不是照念的剧本。访谈哲学、硬规则、挖掘技巧见 `../SKILL.md`。

## 目标
发现人生驱动力。

## 开场白（破冰）
> "有没有什么人，你是打心底佩服的？不是表面的成功，是真的让你服气的那种。"

价值观太抽象，直接问"你的价值观是什么"多半得到空话。从"佩服的人"切入，用具体的人折射价值观。

## 核心问题（地图，按需自然带入，绝不连续抛出）
- 最佩服什么样的人？
- 什么事情最无法接受？
- 成功意味着什么？
- 理想生活是什么样？

单轮只问一个；善用**反例排除**（"绝对不想成为什么样的人"）——讨厌的东西往往是价值观最硬的内核。

## 覆盖检查清单
- [ ] 佩服的人（正向榜样）
- [ ] 无法接受的事（反向边界）
- [ ] 成功的定义
- [ ] 理想生活的样子

## 收尾合成
1. **概览**（叙事）："驱动你的，听起来主要是……"
2. **结构化信号**：价值观关键词
3. **确认**后，套 `../templates/VALUES.md` 写入。
