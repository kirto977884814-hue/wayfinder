# Session 6：关系与影响

> 本文件是访谈者的「地图」，不是照念的剧本。访谈哲学、硬规则、挖掘技巧见 `../SKILL.md`。

## 目标
识别关键关系与影响源。

## 开场白（破冰）
> "你生命中对你影响比较大的人，能想到谁？不一定是名人，身边人也算。"

从具体的"人"切入，关系画像才聊得实。

## 核心问题（地图，按需自然带入，绝不连续抛出）
- 谁是对你影响最大的人？
- 遇到事第一个想跟谁说？
- 谁经常找你帮忙？你反过来找谁？
- 有没有一段关系，改变了你的某个决定？
- 现在最想让谁认可你？

单轮只问一个；"改变决定的关系"和"渴望认可的人"是强信号，值得多挖。

## 覆盖检查清单
- [ ] 影响最大的人
- [ ] 倾诉对象（情感支持）
- [ ] 互助关系（求助 / 被求助）
- [ ] 改变过决定的关系（转折性）
- [ ] 渴望被认可的人

## 收尾合成
1. **概览**（叙事）："你的关系里，主线是……"
2. **结构化信号**：关键关系 / 影响源关键词
3. **确认**后，套 `../templates/RELATIONSHIPS.md` 写入。

## 输出
`RELATIONSHIPS.md`
