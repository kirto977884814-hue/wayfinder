# Session 5：当前阶段分析

> 本文件是访谈者的「地图」，不是照念的剧本。访谈哲学、硬规则、挖掘技巧见 `../SKILL.md`。

## 目标
识别当前成长阶段。

## 开场白（破冰）
> "最近这段时间，你过得怎么样？心里踏实吗，还是有点悬着？"

先做情绪 check-in，让用户感受到被关心、不被评判。迷茫的人往往憋了一肚子话，先接住情绪。

## 核心问题（地图，按需自然带入，绝不连续抛出）
- 目前最大的困惑？
- 最想解决什么问题？
- 未来一年最期待什么？

单轮只问一个；这是最需要**安全感**的一节，多听、多接情绪，少急着给建议（本节只记录，不建议）。

## 覆盖检查清单
- [ ] 当前状态（踏实 / 悬着 / 迷茫 / …）
- [ ] 最大困惑
- [ ] 最想解决的问题
- [ ] 未来一年的期待

## 收尾合成
1. **概览**（叙事）：用一段话描述用户当前处于什么阶段
2. **结构化信号**：`current_stage`（一句话阶段判断）+ 困惑 / 期待
3. **确认**后，套 `../templates/CURRENT_STAGE.md` 写入。

## 输出
`CURRENT_STAGE.md`（同时更新 `PROFILE.md` frontmatter 的 `current_stage`）
